# FIRE Calculator CH

Swiss FIRE calculator backend (Spring Boot)

## Requirements
- Java 17+
- Maven
- SQLite (for local development)

## Setup
1. Copy `src/main/resources/application.properties.example` to `src/main/resources/application.properties`
2. Adjust database configuration if needed
3. Run the project:
   ```bash
   mvn spring-boot:run
